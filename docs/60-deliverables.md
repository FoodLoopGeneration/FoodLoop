# 60 — Deliverables (cosa consegnare)

Nel repository devono essere presenti:

## Codice
- Progetto Spring Boot compilabile ed avviabile
- Struttura a livelli (controller/service/repository/entity)

## Database
- Script o migrazioni (schema + dati demo)
- Admin seed documentato

## Documentazione
- Cartella `/docs` compilata
- Diagrammi (ER + almeno 1 tra Use Case / Class / Sequence)
- README con istruzioni di avvio

## Struttura cartella di release
```
release-GRUPPO_N/
├── configuration/
├── docs/
└── fileeseguibile.jar
```
